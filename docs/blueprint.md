# **App Name**: AlphaBet League

## Core Features:

- User Authentication: Secure user login and logout, utilizing a password system and Supabase for authentication token management.
- Interactive Betting Table: A table interface for users to input and manage their score predictions for multiple football matches within a round.
- Dynamic Match Calendar: Display a browsable calendar of Brasileirão Serie A matches, showing game details, dates, times, and real-time statuses (e.g., Scheduled, Live, Finished).
- Real-time Score Calculation & Ranking: Automatically calculates and updates player points and a leaderboard based on submitted bets and live/final match results. Highlights round winners.
- AI-Powered Bet Assistant Tool: Provide intelligent betting suggestions and insights by analyzing historical team performance, head-to-head records, and current form.
- Admin Visibility Control: An exclusive administrative feature to toggle the visibility of individual player betting scores for other users, enhancing fair play during ongoing rounds.
- Automatic Match Data Synchronization: Continuously fetch and integrate match details, live scores, and final results from an external sports data API into the application, ensuring up-to-date information.

## Style Guidelines:

- Primary color: A vibrant blue (#3A86FF) for trust and a strong connection to Brazilian national colors. This reflects the energetic and competitive spirit of football.
- Background color: A very light desaturated blue-gray (#F4F6F9) that provides a clean, neutral canvas while subtly harmonizing with the primary blue.
- Accent color: An energetic amber (#F59E0B) for call-to-action buttons, highlights, and score badges, providing a vibrant contrast and thematic tie to Brazilian colors (alongside a secondary green for success states).
- Headlines and Body text font: 'Kanit' (sans-serif), chosen for its bold, impactful, and modern appearance, suitable for both prominent titles and readable body content. Note: currently only Google Fonts are supported.
- Utilize a consistent set of modern and easily recognizable icons from Font Awesome to represent actions, statuses, and navigation elements across the application.
- A responsive, content-centered layout that adapts well to various screen sizes, featuring a flexible grid system for displaying multiple match cards and an organized table for score predictions.
- Integrate subtle animations (e.g., pulsing status indicators, sliding transitions for notifications) and smooth element transitions on hover or interaction to enhance user engagement and provide clear feedback.
